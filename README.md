# SRB Grist PWA

A Progressive Web App (PWA) for internal team use, powered by Grist.

## Features
- Connects to self-hosted Grist instances
- Offline capable (PWA)
- Installable on mobile devices
- Secure API key storage (local storage)

## Setup
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`

## Deployment
Automatically deploys to GitHub Pages via GitHub Actions.
